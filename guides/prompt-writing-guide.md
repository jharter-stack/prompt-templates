# Prompt Writing Guide

## Core Rules
- Be specific about the task
- Give the model enough context
- State the target audience
- Define the output format
- Include constraints when needed
- Ask for practical output, not vague fluff

## Strong Prompt Ingredients
- role
- goal
- context
- inputs
- constraints
- output format

## Common Mistakes
- being too vague
- skipping audience details
- asking for too much at once
- failing to define the output structure
- using weak or incomplete inputs
