# Naming Conventions

## Use
- lowercase
- hyphens
- one clear purpose per file

## Good Examples
- seo-brief-builder.md
- meeting-summary.md
- competitor-analysis.md
- workflow-template.md

## Avoid
- final-v2-real.md
- new-best-prompt.md
- stuff.md

## Rule
Name files so someone can understand them at a glance.
